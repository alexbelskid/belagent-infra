# belagent.com — UI Concept

## Главная идея

Две версии интерфейса из одной кодовой базы:
- **Business** — для клиентов (Григорий, Lemuel, Евгений)
- **Personal** — для тебя (полный контроль, все настройки)

Переключатель в углу. Один и тот же дашборд, разный набор вкладок.

---

## Структура (Business mode — для клиента)

### 1. 💬 Chat
Просто чат с ботом. Ничего лишнего.
Клиент не видит что там Claude, токены, модели — просто пишет.

### 2. 📊 Activity Feed
Лента всего что делает бот:
- "Отправил ответ Анне в 14:32"
- "Получил 3 новых заявки"
- "Запустил крон задачу: обновление базы"
- "Сохранил в память: Иван Петров, новый клиент"

Как Instagram feed — прокручиваешь вниз, видишь историю.

### 3. ⚡ Automations (крон)
Список активных автоматизаций:
```
[▶] Ежедневный отчёт        каждый день в 09:00   ✅ Работает
[▶] Напоминание клиентам    каждые 3 дня           ✅ Работает
[⏸] Парсинг заявок          каждый час             ⏸ Пауза
[+] Добавить автоматизацию
```
Кнопки: Запустить / Пауза / Удалить. Никакого cron синтаксиса.

### 4. 🧠 Memory
Карточки того что бот знает:
```
👤 Клиенты (12)     📁 Документы (5)     📌 Заметки (8)
```
Открыть карточку → редактировать прямо в UI.

### 5. 🔌 Connections
Плитки подключённых сервисов:
```
✅ Telegram          ✅ Google Workspace    ❌ Instagram
✅ Email (Gmail)     ❌ WhatsApp            + Добавить
```
Клик → настройки подключения.

---

## Personal mode (для тебя)

Те же вкладки + дополнительно:
- **Sessions** — история сессий, токены, стоимость
- **Config** — все настройки OpenClaw
- **Logs** — raw логи
- **Agents** — управление несколькими агентами
- **Nodes** — подключённые устройства

---

## Ключевые принципы UX

1. **Нет модел/провайдеров в интерфейсе клиента** — только результат
2. **Всё на русском** для клиентов-россиян/беларусов
3. **Mobile-first** — большинство смотрят с телефона
4. **Activity Feed вместо логов** — понятно что происходит
5. **One-click actions** — запустить задачу, добавить напоминание

## Технический план

```
openclaw/ui (форк)
  ├── src/ui/views/
  │   ├── activity-feed.ts      ← НОВОЕ
  │   ├── automations.ts        ← упрощённый cron.ts
  │   ├── memory-cards.ts       ← НОВОЕ
  │   └── connections.ts        ← НОВОЕ
  └── src/ui/
      ├── app-mode.ts           ← business/personal switch
      └── navigation.ts         ← разные наборы вкладок
```

**Данные:**
- Activity Feed ← sessions.jsonl (парсим роли user/assistant)
- Automations ← WS: cron.list, cron.add, cron.remove
- Memory ← workspace/memory/*.md (file API)
- Connections ← config channels + custom plugins
