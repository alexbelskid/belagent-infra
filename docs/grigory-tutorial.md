# 🤖 Как подключиться к своему AI-агенту

## Шаг 1 — Открыть ссылку

Перейди по ссылке:

👉 **https://grigory.belagent.com**

---

## Шаг 2 — Открыть ссылку с токеном

Токен передаётся прямо в ссылке (после `#tok=`). Используй ссылку, которую тебе прислал оператор — она выглядит так:

👉 **https://grigory.belagent.com/#tok=ВАШ_ТОКЕН**

Добавь в закладки! Токен в хэше (#) не отправляется на сервер — остаётся только в браузере.

---

## Что доступно

- 💬 **Chat** — чат с ботом
- 📊 **Overview** — общая статистика
- ⚡ **Cron** — автоматические задачи
- 🔌 **Connections** — подключённые сервисы (Telegram, Gmail и др.)
- 🔄 **Activity** — лента событий
